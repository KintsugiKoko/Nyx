# Nyx Game Design Documents

This folder holds game design documents for Nyx. These notes describe the intended player experience, loops, systems, and content direction.

Nyx is still a work-in-progress Unreal C++ prototype. GDD notes should stay honest about what is implemented, what is planned, and what still needs testing.

## Documents

- [Core Game GDD](Nyx_Core_GDD.md) - high-level experience, pillars, loops, FTUE, merchant, Koi skills, and progression direction.

## GDD Rules

- Keep the player experience readable before adding detail.
- Mark prototype systems as WIP.
- Do not claim final art, final tuning, or complete content.
- Use real class names only when they already exist in `Source/Nyx`.
- Explain why a feature exists, not only what it does.
